Ifun tem como objetivo levar o usuário para uma aventura além da imaginação. É um jogo 2D criado pelo projeto "8Bits". Tem como protagonistas: Yohannã, Thiagod e Gabrielfo. O Objetivo do jogo é derrotar todos os seus inimigos, para assim, o grande churrasco real ocorrer.
